# Web Traffic Analytics

## Project Overview
This project analyzes website traffic data to understand user behavior, identify key metrics, and uncover insights to improve engagement and conversions.

## Technologies Used
- Python 3
- Pandas - Data cleaning and analysis
- Matplotlib / Seaborn - Data visualization
- NumPy - Numerical computations

## Features
1. Data Cleaning - Removed nulls, duplicates, formatted dates
2. Exploratory Data Analysis - Sessions, bounce rate, conversions, sources
3. Data Visualization - 8 charts covering all key metrics
4. User Journey Mapping - Drop-off and engagement analysis

## Dataset
Web_Traffic_Data.csv — 5,000 website session records with page views, session duration, traffic sources, device types, and conversion data.

## Key Metrics
- Total Sessions: 5,000
- Total Page Views: 37,534
- Average Session Duration: 2.90 minutes
- Bounce Rate: 6.80%
- Conversion Rate: 7.58%
- New Users: 59.96%

## Key Findings
- Organic Search is the top traffic source (36%)
- Mobile users make up 47.6% of all sessions
- Home and Products pages get the most traffic
- India and USA are the top countries by sessions
- 60% of visitors are new users

## How to Run
pip install pandas matplotlib seaborn numpy
python web_traffic_analysis.py

## Outcome
Identified key user behavior patterns and drop-off points. Provided actionable recommendations to improve mobile experience, SEO strategy, and user retention.
